# 判断图中是否有环

## 有向图

### 1. 基于度的算法

（类似于有向图的拓扑排序）

__算法步骤：__

```
1. 求出所有顶点的度
2. 若有度 <=1 的顶点，转 3, 否则转4
3. 删除所有度 <=1 的顶点及其边，并将与这些边相连的顶点的度-1
4. 若图中仍有顶点，则说明有环。否则无环。
```

__时间复杂度:__  O(E + V). 其中 E, V 分别是边和顶点的数目
